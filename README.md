**Coffee Order Excel Dashboard**
**Project Overview**
This project focuses on analyzing coffee sales data using Microsoft Excel. 
The objective was to transform raw transactional data into meaningful business insights through data cleaning,aggregation,and visualization.
The dashboard provides a clear summary of sales performance,customer contribution, product trends, and country-wise analysis.

**Dashboard Features**
- Total Sales Analysis
- Top 5 Customers by Revenue
- Sales by Country
- Product Performance Overview
- Order-Level Data Analysis

**Tools & Techniques Used**
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Data Aggregation
- Charts & Visualization

- **Key Business Insights**
- Identified highest revenue-generating customers
- Analyzed country-wise sales contribution
- Compared product performance trends
- Structured raw order data into actionable insights
